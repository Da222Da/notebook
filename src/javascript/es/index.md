# ECMAScript 语法标准

ECMAScript 是 JavaScript 脚本语言的语法标准，由 W3C 国际组织定义。

如果你想了解更多信息的话，请移步 [MDN](https://developer.mozilla.org/zh-CN/docs/Learn/JavaScript)。
