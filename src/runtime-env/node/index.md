# NodeJS 是什么？

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行时环境，让 JavaScript 拥有了操作系统进行交互的能力。

如果你想了解更多关于 NodeJS 的信息的话，请移步 [https://nodejs.org](https://nodejs.org).
